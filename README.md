本项目是一个vue-h5的模板项目,已经配好了一些移动端开发的依赖和一些loader.方便快速开发

### 安装依赖
npm install or cnpm install (推荐使用yarn安装,更快)

# run in dev
npm run dev

# build
npm run build

***
目前已经集成的如下所示:
- `vuex`
- `vue-router`
- `px2rem loader`
- `fastclick`
- `less loader`
- `axios`
- `qs`
- `moment`
- `babel-polyfill`
- `babel-plugin-import`
